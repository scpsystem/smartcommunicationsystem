Tetapan Tag NFC

Jana kandungan untuk ditulis ke dalam tag NFC menggunakan aplikasi NFC Tools atau seumpamanya.

⚙️ Cara menulis tag:
  1. Muat turun app NFC Tools pada telefon Android/iOS
  2. Buka app → pilih "Write" → "Add a record" → "Text"
  3. Salin payload dari bawah dan tampal ke dalam medan teks
  4. Sentuh telefon ke atas tag NFC untuk menulis
Format: Teks Mudah
